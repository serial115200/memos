mainline 内核更新
================================================================================

软件安装

.. code-block::

    sudo add-apt-repository ppa:cappelikan/ppa
    sudo apt update && sudo apt install mainline -y


内核安装

.. code-block::

    sudo mainline --install-latest


其它事项可以查看 mainline 的帮助信息，善用 uname -r 检查版本


系统重启

.. code-block::

    sudo reboot
