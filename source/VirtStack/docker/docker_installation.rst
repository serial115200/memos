Docker 安装
================================================================================

Docker 安装参考:

    https://docs.docker.com/engine/install


Ubuntu 也按照以下方式安装：

.. code-block:: bash

    sudo apt update
    sudo apt install docker.io
    sudo systemctl restart docker


Docker 官方文档说 Ubuntu 的 docker.io 过时了，但最近社区又开始维护。


Docker 软件包存在大量历史遗留问题，详见 docker的版本.mhtml
