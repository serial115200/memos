QEMU 安装
================================================================================

基础软件

.. code-block:: shell

    sudo apt update
    sudo apt install qemu-system qemu-user qemu-utils

硬件加速

.. code-block:: shell

    sudo apt install qemu-kvm

网桥模块

.. code-block:: shell

    sudo apt install bridge-utils
